## Errors, Aging Items, and Blockers

### Test Failures (10 + 1 error)

#### Category 1: Python 3.11 f-string SyntaxError (4 tests) — BLOCKER for CLI import

`interactive.py:4097` contains `\"` inside an f-string, which is invalid in Python < 3.12.

```
SyntaxError: f-string expression part cannot include a backslash
  File "income_desk/cli/interactive.py", line 4097
    print(f"  {_styled('positions = [PositionExposure(ticker=\"SPY\", ...)]', 'dim')}")
```

**Affected tests:** `test_cli_functional.py` (2), `test_setup.py` (2)
**Impact:** Any test that imports `income_desk.cli.interactive` fails on Python 3.11.
**Fix:** Replace `\"` with `'` or use a string variable outside the f-string.

#### Category 2: Network/Proxy blocked (6 tests + 1 error) — ENVIRONMENT ONLY

yfinance cannot connect (`curl: CONNECT tunnel failed, response 403`). Tests that fall through to live data fetch fail.

**Affected tests:**

**Impact:** Environment-specific — passes in normal network environments.
**Note:** `test_release_readiness.py` also fails because BANKNIFTY simulated data path falls through to live yfinance fetch.

#### Category 3: Floating point precision (1 test)

```
test_position_sizing.py::TestPairwiseCorrelation::test_zero_variance_returns_zero
```

**Impact:** Minor — numerical epsilon issue, not a trading risk.
