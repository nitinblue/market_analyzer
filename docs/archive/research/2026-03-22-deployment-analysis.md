### V. GAPS IN CURRENT ANALYSIS

This analysis was limited by:

1. **India IV data unavailable** — without Dhan broker connected, we cannot assess premium quality for India tickers. The recommendation to trade TCS credit spread assumes adequate premium exists. This MUST be verified with live quotes before execution.

2. **India crash sentinel not built** — the sentinel uses US tickers. An India-specific version using NIFTY/BANKNIFTY/India VIX would provide more accurate signals for India deployment timing.

3. **No regime transition alerting** — the deployment plan is triggered by regime changes, but income-desk has no persistent alert for "notify me when NIFTY exits R4." This must be checked manually each morning.

4. **Cross-market correlation not quantified** — we assert "India is worse than US" based on regime labels but haven't quantified the current US→India correlation or predicted the India recovery lag.

5. **Earnings calendar not factored** — India Q4 FY26 results begin in April. Positions entered in Phase 1-2 may overlap with earnings season. The earnings blackout gate handles individual trades but not the broader portfolio timing.
